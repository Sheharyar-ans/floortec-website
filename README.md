## Installation

To set up the Floortec website locally, follow these steps:

### Prerequisites

- A local server environment (e.g., XAMPP, WAMP, MAMP)
- PHP version 7.4 or higher
- MySQL version 5.7 or higher
- Git installed on your machine

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/YourUsername/floortec-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd floortec-website
   ```

3. Set up a new MySQL database for WordPress.

4. Rename the `wp-config-sample.php` file to `wp-config.php` and update the database details.

5. Install WordPress by navigating to `http://localhost/floortec-website` in your web browser.

6. Follow the on-screen instructions to finish the installation.

7. Activate the Floortec theme and any required plugins.

## Usage

After installation, you can access the WordPress admin panel at `http://localhost/floortec-website/wp-admin`. From there, you can manage the website content, including pages, posts, and media library, as well as modify settings and install additional plugins.
