# Floortec Website

## Overview
Floortec Website is a WordPress project for the company Floortec. This website is designed to showcase the company's products and services related to flooring solutions.

## Features
- Responsive design for optimal viewing on all devices
- Integration with social media platforms
- Contact forms for easy communication with customers
- Gallery to showcase the company's work and products
- SEO optimization for better visibility on search engines
